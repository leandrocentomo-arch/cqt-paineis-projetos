# cqt-paineis-clientes

Painéis operacionais estáticos da CQT, hospedados via GitHub Pages porque o compartilhamento de Artifact do Claude exige plano Team/Enterprise.

- `painel-sodexo.html` — agenda e notas fiscais compartilhado com a Sodexo (Vanessa Borba). Listado em `index.html`.
- `painel-rubens.html` — dashboard financeiro interno (repasses, notas fiscais) pro Rubens Centomo. **Não** listado em `index.html` de propósito (dado mais sensível), mas o repositório é público: qualquer um que abra a página do repositório no GitHub vê o arquivo na lista normalmente. Isso não é controle de acesso, é só discrição de não deixar o link visível na home.

Cada arquivo é HTML autocontido (CSS/JS/fontes embutidos), atualizado manualmente pelo Leandro via Claude Code. Não é atualizado automaticamente.
